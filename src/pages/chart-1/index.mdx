# 攻略チャート 1 オープニング〜Z

## 1. オープニング〜

### オープニング

オープニング後、イベント中に「オウルベア」と戦闘となります。その後はクレスタへフェーズが移り、ロニが「いい兄貴」の称号を獲得します。

<Tips title="アミィの贈り物">
  {
    <p>
      南の森へ狩りに行く前に、チェスターの家2階にいる妹のアミィに話しかけると「マスコット」「チャネリング」「アップルグミ」がもらえます。（チャネリングはPSP版では入手できない。）
    </p>
  }
</Tips>
